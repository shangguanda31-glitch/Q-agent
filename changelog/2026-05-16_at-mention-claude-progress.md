# 修复 @识别 + Claude Code 进度通知

**日期**: 2026-05-16
**类型**: Bug 修复

## 变更内容
- @消息显示区分：[@我] = @机器人自己，[@xxx] = @别人，[@所有人] = @all
- 系统提示词明确告知 LLM [@我] 才需要响应，[@号码] 是别人
- 新增进度反馈规则：claude_code 前/后必须 notify_user

## 原因
- 所有 @ 都显示为 [@]，LLM 不知道是否在 @自己
- Claude Code 执行长时间任务时用户无任何反馈

## 变更文件
- src/agent/mod.rs（@消息显示区分自己和别人）
- src/agent/prompt.rs（新增 @识别规则和进度反馈规则）
