# 修复 LLM 错误响应 @别人的消息

**日期**: 2026-05-16
**类型**: Bug 修复

## 变更内容
- 新增「判断消息是否与自己相关」章节，明确规则：
  - `[@我]` → 需要关注
  - `[@数字]` → @了别人，忽略
- 移除通知章节中关于 @别人可忽略的旧规则，统一到独立章节

## 原因
- 群消息 `[@2159682228] 你给我写个报告` 中 @的是别人，但 LLM 看到"你给我写"就以为是给自己的指令
- LLM 错误地调用了 notify_user 和 claude_code

## 变更文件
- src/agent/prompt.rs（新增独立章节判断消息是否与自己相关）
