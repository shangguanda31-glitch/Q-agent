# 修复上下文压缩（阈值调低 + 中文 token 估算修正）

**日期**: 2026-05-16
**类型**: Bug 修复

## 变更内容
- 上下文压缩阈值从 6144 降至 4096，提前触发避免超出模型窗口
- token 估算改用 `byte_len * 2 / 5`，对中文更准确（原 `byte_len / 4` 低估了中文 token 数，中文每字约 1.5-2 token）

## 原因
- 中文 token 估算不准确导致压缩触发太晚，模型上下文已超限
- 超限后摘要 LLM 调用也失败，进入死锁状态

## 变更文件
- src/agent/mod.rs（MAX_TOKENS 6144→4096，len/4 → len*2/5）
